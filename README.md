# Mina Lidya - 4 Mevsim

Mina Lidya interaktif dijital sanat projesi. Modern web teknolojileri, Three.js ve kreatif tasarım süreçleri ile oluşturulmuş 4 mevsimi konu alan interaktif bir deneyim.

## Özellikler

- Three.js tabanlı interaktif 3D deneyimi
- GSAP ile pürüzsüz animasyonlar
- Optimize edilmiş web varlıkları ve yüksek performans
- Mobil uyumlu tasarım (R3F)

## Geliştirme Süreci

Bu proje Vite tabanlı bir React uygulamasıdır.

### Kurulum

```bash
npm install
```

### Yerel Geliştirme

```bash
npm run dev
```

### Prodüksiyon Build

```bash
npm run build
```
